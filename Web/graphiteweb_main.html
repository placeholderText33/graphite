graphite.

Graphite is a web based physics simulation tool for modelling dynamics.

The physics engine is capable of simulating:

  • Momentum and Impulse
  • Elastic/Inelastic Collisions
  • Gravity
  • Friction

Once you click download button, you'll be taken to the code. Click the green code button and download the zip file. Extract the zip file on your machine and run "engine.html" to use the program.
